Ferramentas Utilizadas

(Voltar ao topo)

A ferramenta principal que será utilizada no decorrer do curso será o Visual Studio Code, sendo está uma ferramenta open source* ou seja, uma ferramenta gratuita de código aberto.

Para desenvolvimento de paginas web, podemos utilizar qualquer editor de texto simples, como Bloco de notas, Sublime, Notepad++ , dentre outros disponíveis, vae da preferência.

O Visual Studio Code pode ser baixado através da página Visual Studio Code.*

Usando o Inspetor de Elementos

(Voltar ao topo)

Uma das ferramentas que podemos utilizar para auxiliar no desenvolvimento dos projetos front-end, é o inspetor de elementos, uma ferramena muito útil para o desenvolvedor.

Quem oferece essa ferranenta para nós é o navegador utilizado nos projetos, para acessá-la, com o navegador aberto, apertamos o F12 no teclado, e o inspertor de elementos é exibido, com várias informações do site em que o navegador está aberto. Outra forma possível de abrir essa ferramenta é clicando com o direito do mouse sobre a página aberta no navegador e clicando em inspecionar elemento ou inspecionar, dependendo do navegador.

Nessa ferramenta, possuímos vários recursos para analisar como a página foi feita, ou seja todo o código HTML/CSS foi formatado na página.

Estrutura Básica do HTML

(Voltar ao topo)

A cada projeto que iniciamos no Visual Studio Code, uma boa prática a se fazer, é criar uma pasta específica para cada projeto criado. Devemos abrir a pasta clicando em arquivo, abrir pasta e selecionar a pasta do projeto em questão, dessa forma toda a estrutura do projeto será exibida na árvore do lado esquerdo do Visual Studio, então toda mudança, ou atualização desse projeto deverá ser salvo na sua respectiva pasta, dessa forma podemos deixar tudo mais organizado e de fácil acesso.

A consistência de uma página HTML possui basicamente 3 partes, as chamadas 'Tags', que são:

  • html
  • head
  • body

Falando Sobre Tags

(Voltar ao topo)
  • html: Tudo que estiver definido dentro destas 'Tas', define para o navegador que esta é uma página html, possuindo a seguinte característica:

    • < html> : Dessa forma, a 'Tag' foi aberta

    • < /html> : Dessa forma, a 'Tag' foi fechada
  • head: Tudo que estiver definido dentro destas 'Tas', seria o cabeçalho da página, pré-carrega informações antes de exibir para o usuário, possuindo a seguinte característica:

    • < head> : Dessa forma, a 'Tag' foi aberta

    • < /head> : Dessa forma, a 'Tag' foi fechada
  • body: Tudo que estiver definido dentro destas 'Tas', define oque de fato, será exibido para o usuário, possuindo a seguinte característica:

    • < body> : Dessa forma, a 'Tag' foi aberta

    • < /body> : Dessa forma, a 'Tag' foi fechada

Atributos Básicos

(Voltar ao topo)

Atributos das 'Tags' nada mais é, doque as propriedades que as 'Tags' podem ter, podendo ser atributos globais, ou seja , que todas as 'Tags' vão ter, e os atributos específicos de cada 'Tag'.

  • São atributos globais/genéricos:

    • id: Identificador de algum elemento que se deseja alterar, muito utilizado no javascript.
    • style: Pode ser utilizado para adicionar formatação diretamente no elemento da página html, porém é voltado exclusivamente para formatação via CSS.
    • class: Podendo ser adicionado também para formatação direta no elemento html, é voltado exclusivamente para formatação via CSS.
  • type: O atributo input, é utilizado para criar campos de formulário , e esse atributo 'TYPE' é específico do atributo 'INPUT', podendo receber valores como 'text', 'number', 'color', dentre outros.

  • src: Utilizado para se adicionar o endereço de localização das imagens que serão adicionadas na página, é específico do atributo 'IMG', podendo estas imagens serem, imagens salvas na pasta do projeto, ou um link de uma página externa.

Executando Uma Página HTML

(Voltar ao topo)

Para executar as páginas diretamente no Visual Studio Code, podemos utilizar uma extensão criada especificamente para isso, é só instalar o 'Live Server'. Para executar a ferramenta, devemos clicar com o direito do mouse sobre o arquivo html e abrir a opção 'Open With Live Server', com isso uma página do navegador padrão executará o projeto.

Listas Ordenadas e Não Ordenadas

(Voltar ao topo)

Basicamente, existem 2 tipos de listas que são utilizadas na criação de uma página html, elas seguem um padrão pré-definido pelo navegador, porém podemos estiliza-las via CSS.

Listas ordenadas, seguem um padrão numérico para identificação dos itens, como exemplo a seguir:
  1. Lista ordenada
  2. Lista ordenada
  3. Lista ordenada
Listas não ordenadas, seguem um padrão simbólico como pequenos pontos, sendo este o padrão do navegador para identificação dos itens, como exemplo a seguir:
  • Lista não ordenada
  • Lista não ordenada
  • Lista não ordenada

Links

(Voltar ao topo)

Toda página que utilizamos hoje em dia, possui um link que pode te direcionar para outra página externa, ou nos direciona para outra aba do mesmo site.

Para utilizarmos essa fincionalidade utilizaremos a 'Tag' < a> (abre 'Tag'), < /a> (fecha 'Tag') , que significa "ancoragem", ou seja, fazer a ligação/ancoragem entre o link digitado e a página que se deseja visitar quando clicado.

  • Exemplo:

    • Dio.me
